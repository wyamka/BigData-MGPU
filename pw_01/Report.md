## Введение

В рамках практической работы рассматриваются принципы функционирования распределённой файловой системы HDFS и механизмы аналитической обработки данных с использованием PySpark. Практическая часть направлена на решение прикладной бизнес-задачи, включающей этапы загрузки данных, их очистки, агрегации, выполнения SQL-запросов и последующей визуализации результатов для поддержки управленческих решений.

Целью работы является освоение базовых принципов работы с распределённой файловой системой HDFS и фреймворком Apache Spark, а также формирование практических навыков загрузки, обработки, анализа и визуализации больших объёмов данных с использованием PySpark и Spark SQL для решения прикладных бизнес-задач.
